**Tarea Java Swing**

– Simulador Bancario Interactivo

Objetivo:

Comprender y realizar una simulación de un sistema bancario utilizando Swing Java Design

**Inicio(Login)**

El sistema Bancario al inicio de su ejecución muestra la ventana del Login la cual permite que el usuario ingrese al sistema. 

<img width="430" height="264" alt="image" src="https://github.com/user-attachments/assets/adea74fb-19f1-4d9b-9bea-4265877ba5fb" />


**Ventana Principal BancoForm**

Después de dar click en el botón de ingresar se despliegan las opciones de transacciones que puede realizar el usuario que son las siguientes:

•	Depósito 

•	Retiro 

•	Transferencia 

•	Salir 

Cada una le permite realizar distintas acciones según lo que quiera el usuario.
 
 <img width="589" height="432" alt="image" src="https://github.com/user-attachments/assets/d109df6d-a814-4c30-a671-b929585a4671" />


Además, está el historial de transacciones que permitirá ver qué acciones ha realizado el usuario. 

**Opción de Depósito**

En la opción de Depósito va a permitir depositar un valor y después saldrá un mensaje de confirmación esto se sumará al saldo inicial que tenga el usuario.

**Opción de Retiro**

En la siguiente opción que es la de Retiro esto permitirá ingresar el valor a retirar y saldrá un mensaje de confinación siempre y cuando el retiro sea válido , además ese monto ingresado se descontara del saldo del usuario.

**Opción de Transferencia**

La tercera opción que es de Transferencia el sistema permitirá ingresar el nombre a la persona que se desea trasferir y el monto, si la transacción es valida se la realizara con éxito mostrando un mensaje de confirmación y descontando el monto transferido en el saldo del usuario.

**Opcion Salida**

La última opción de salida permitirá finalizar la ejecución del programa con una ventana que confirmara que se eligió esta opción al usuario. 
Para finalizar el esta Simulación del Sistema Bancario es muy fácil de utilizar ya que son procesos concretos y en la interfaz nos da a entender cada acción que se va a realizar.
